jQuery 9-way direction control.

Directions
----------
n, s, w, e, ne, nw, se, sw, c

Options
-------
values	= { 'n': 'top left' }
disable = ['w', 'e']
hide	= ['c']
style	= 'carat', 'triangle', 'arrow', 'arrowthick' or a plainobject or undefined for default ('arrow')
